### 2. Формат catalog.json (бот полностью управляет файлом)

```JSON
{
  "items": [
    {
      "id": "brooch-0001",
      "title": "Брошь 'Морской бриз'",
      "description": "Ручная работа из японского бисера...",
      "price": 1250,
      "stock": 3,
      "status": "available",          // available / sold / saved
      "created_at": "2026-03-17",
      "updated_at": "2026-03-17",
      "images": ["brooch-0001-01-main.jpg", "brooch-0001-02-side.jpg"]
    }
  ]
}
```

### 3. Инженерия и Запись данных (Критично)

- GitHub REST API: Бот не использует git clone или git push. Все изменения в catalog.json отправляются прямыми PATCH/PUT запросами через GitHub API (например, через PyGithub).
- Блокировка потоков (Lock): В боте используется asyncio.Lock(). При одновременном нажатии кнопки "Заказать" двумя покупателями, запросы выстраиваются в очередь. Это исключает конфликт версий catalog.json.
- Сортировка: По полю updated_at. Админ может «поднять» товар кнопкой — бот обновляет updated_at на текущее время.

#### Для админа (появляется автоматически по user_id):
	
- Добавить товар

- Активные заказы
- Готово

### 6. Процесс заказа

- Покупатель нажимает «Заказать эту брошь».
- Бот собирает данные (ФИО, телефон, адрес).
- Очередь asyncio.Lock() блокирует файл -> Бот проверяет наличие -> Уменьшает stock -> Меняет status на "sold" при stock == 0.
- Сохраняет заказ локально или отправляет в отдельный data/orders.json через API.
- Сразу отправляет уведомление тебе и менеджеру (Анастасии) в личку.

### 10. Frontend (site/)

Сайт представляет собой статическое одностраничное приложение, созданное с использованием HTML, Tailwind CSS и нативного JavaScript.

-   **Динамический каталог:** Товары асинхронно загружаются из файла `catalog/catalog.json`.
-   **Интерактивная галерея:** Каждая карточка товара оснащена интерактивной галереей изображений:
    -   **Навигация:** Поддерживается переключение слайдов с помощью стрелок, точек-индикаторов и клавиатуры (в полноэкранном режиме).
    -   **Автопроигрывание:** При наведении курсора автоматически запускается слайд-шоу с 6-секундным интервалом, которое приостанавливается при взаимодействии пользователя. Прогресс-бар показывает время до следующего слайда.
    -   **Полноэкранный режим:** Изображения можно просматривать в полноэкранном режиме с сохранением всей навигации.
-   **Адаптивный дизайн:** Макет полностью адаптивен и корректно отображается на экранах разных размеров.
